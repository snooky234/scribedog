# Changelog

All notable changes to ScribeDog are documented here. Format loosely follows
[Keep a Changelog](https://keepachangelog.com/).

## [Unreleased]

### Highlights
- **Barkdown is now ScribeDog!** Same app, same dog, new name — renamed again
  to avoid a naming conflict with an existing trademark. **Note for existing
  users:** ScribeDog installs alongside the old Barkdown app; please uninstall
  Barkdown manually and re-select your settings once. Your documents are plain
  Markdown files and are not affected.
